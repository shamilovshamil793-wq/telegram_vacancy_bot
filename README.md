# 🤖 Telegram бот для поиска вакансий на hh.ru

Бот ищет вакансии через официальное API hh.ru.

## Команды

- `/start` — приветствие
- `/help` — помощь
- `/popular` — популярные запросы с кнопками

## Технологии

- Python 3.12
- python-telegram-bot
- requests
- hh.ru API

## Установка и запуск

1. Клонируйте репозиторий
2. Создайте виртуальное окружение
3. Установите зависимости: `pip install -r requirements.txt`
4. Создайте файл `.env` с вашим токеном
5. Запустите бота: `python bot.py`
